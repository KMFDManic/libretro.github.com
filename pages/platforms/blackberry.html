RetroArch Blackberry

This is a port of RetroArch for Blackberry 10 and Blackberry Playbook Tablet OS. RetroArch Blackberry was first released to the public on 31 May 2013 [version 0.9.9].

What do I need to use it on Blackberry 10/Playbook?

In order to use RetroArch Blackberry, you must sideload the application file (a file with the extension .BAR in this case). This file can be downloaded below.

Blackberry World has similar restrictions as iTunes Store so we can't publish RetroArch Blackberry on there unless they change their arbitrary restrictional policies.
