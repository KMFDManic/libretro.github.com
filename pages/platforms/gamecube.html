RetroArch GameCube

This is a port of RetroArch for the Nintendo GameCube games console. RetroArch GameCube was first released to the public on 17 February 2013 [version 0.9.8]. It is currently at [version 0.9.9].

What do I need to use it on my GameCube?

In order to use RetroArch GameCube, you must have a GameCube games console that has been modified to run unsigned content.

Either one of the following would do:

  • A GameCube 'soft-modified' with an exploitable game (Phantasy Star Online Episode I & II)
    • This requires that you own a Gamecube Broadband Adaptor and (optionally) an SD Gecko adaptor.
  • A retail console hard-modified through a commercial modchip
