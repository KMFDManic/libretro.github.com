RetroArch PS3

This is a port of RetroArch for the PlayStation3 games console. RetroArch PS3 was first released to the public on 3 June 2012 [version 0.9.6]. It is currently at [version 0.9.9].

What do I need to use it on my PlayStation3?

In order to use RetroArch PS3, you must have a PlayStation3 games console that has been jailbroken and/or is running a custom firmware so that you can run unsigned content.

Either one of the following would do:

  • A debug console (DEX/DECR)
  • A retail console jailbroken (CEX) with a PSJailbreak/PSGroove dongle [up to firmware 3.41]
  • A retail console jailbroken (CEX) with one of the various custom firmwares [up to firmware 3.55]

It is not possible to run RetroArch PS3 on a PlayStation3 games console that is dependent on Cobra ODE or something similar like that - since it can't run unsigned content.
