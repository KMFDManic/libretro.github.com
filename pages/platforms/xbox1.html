RetroArch Xbox1

This is a port of RetroArch for the original Xbox games console. RetroArch Xbox1 was first released to the public on 23 August 2012 [version 0.9.7]. It is currently at [version 0.9.9].

What do I need to use it on my Xbox 1?

In order to use RetroArch Xbox1, you must have an Xbox1 games console that has been modified to run unsigned content.

Either one of the following would do:

  • A debug console (XDK)
  • An Xbox 'soft-modified' with an exploitable game (Mech Assault/Agent Under Fire/Splinter Cell)
  • A retail console hard-modified through a commercial modchip

Where to get it?

You can find the most recent version at "the usual places".
