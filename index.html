What is RetroArch?

RetroArch is a modular multi-system game/emulator system that is designed to be fast, lightweight, and portable. It has features few other dedicated games / emulators have, such as:

  • Real-time rewinding
  • Multi-pass shaders (up to 8-pass)
  • Game aware shading
  • Remote networked sound support
  • A built-in GUI/OSD system
And more...

What platforms does it support?

RetroArch sets the standard on cross-platform portability right now.

  • Linux (x86 and x86_64)
  • Windows (32bit/64bit)
  • Mac OSX (PPC/Intel)
  • Playstation 3 (PS3 SDK/PSL1GHT (WIP))
  • Xbox 1 (XDK)
  • Xbox 360 (XeXDK/Libxenon)
  • Gamecube
  • Wii
  • Raspberry Pi
  • OpenPandora
  • Android
  • Blackberry (10/Playbook)
  • iOS
